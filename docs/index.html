¿Qué son las criptomonedas?

Las criptomonedas son monedas digitales o virtuales que utilizan tecnología criptográfica para garantizar la seguridad de las transacciones.

  • Descentralización: No dependen de bancos centrales.
  • Seguridad: Utilizan tecnología blockchain.
  • Transparencia: Todas las transacciones son públicas.
  • Accesibilidad: Se pueden enviar y recibir globalmente.
